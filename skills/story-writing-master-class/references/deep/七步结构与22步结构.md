# 七大关键步骤与二十二步结构

## 定义

**七大关键步骤**是特鲁比提出的故事骨架——从开头到结尾必经的七个发展阶段：弱点/需求→欲望→对手→计划→对决→真实自我揭露→新的平衡点。这七个步骤不是外部强加的机械结构，而是以人的行动为基础、内在于故事之中的"DNA"。

**二十二步结构**是在七大步骤基础上，为故事中段(大部分作者失手处)增加15个步骤而形成的完整情节地图。步骤1-3在开头附近，4-18在故事中段，19-22在结尾附近。步骤数量可根据故事长度伸缩(如手风琴)，最少不能少于七个。

## 判定标准

- 故事骨架是否围绕"主角的内在转变"组织，而非"外在的幕结构"
- 是否有明确的"弱点/需求"作为故事源泉
- "真实自我揭露"是否通过行动(而非说教)来呈现
- 道德层面的需求/揭露是否与心理层面并存
- 22步的结构标签是否能标注在每个场景上

## 常见错误

1. **混淆需求与欲望**：需求是克服内在弱点，欲望是外在目标。两者是不同步骤。
2. **以欲望开场**：跳过弱点/需求直接写欲望，牺牲了故事根基和转变可能性。
3. **让主角在开头就知道自己的需求**：如果主角一开始就知道自己的需求，故事就结束了。
4. **生搬22步的全部步骤**：步骤数量和顺序因故事而异，不是固定模板。
5. **"对手=坏蛋"的思维**：真正对手是与主角竞逐同一目标的人，主角可能不恨他。

## 交叉引用：McKee 五部分结构

| 维度 | Truby 7步/22步 | McKee 五部分 |
|------|---------------|-------------|
| 结构导向 | 主角内在转变驱动 | 事件因果链驱动 |
| 开头 | 弱点/需求→欲望→触发事件 | 激励事件→进展纠葛 |
| 中段 | 多重揭露/抉择/看似落败/体验死亡 | 进展纠葛→危机 |
| 结尾 | 对决→自我揭露→道德抉择→新平衡 | 高潮→结局 |
| 分类维度 | 步骤精细(22步) | 部分宏观(5部分) |
| 对中段的处理 | 15个步骤详细指导 | 危机→高潮为主 |
| 核心差异 | "幽灵"概念(反欲望机制) | "鸿沟"原理(预期与现实差距) |

两者互补使用效果最佳：用McKee的因果链确保情节动力，用Truby的22步确保角色内在发展的完整性。
