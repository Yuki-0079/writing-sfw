# 道德论点（Moral Argument）

## 定义

道德论点是特鲁比用来替代"主题"或"信息"的概念。它不是题材（死亡、善恶、救赎等），而是作者关于**如何待人处世的独特看法**（道德愿景）。出色的故事通过**结构**而非说教来传达道德论点——结构收敛时主题在观众脑海中扩展。

道德论点通过行动而非对白来表现，具体体现在以下序列中：
1. 主角的信念/价值取向 → 2. 道德层面的弱点 → 3. 道德需求 → 4. 第一次不道德行动 → 5. 欲望 → 6. 驱动力 → 7. 一系列不道德行动+批判+辩白 → 8. 盟友的攻击 → 9. 执迷不悟的驱动力 → 10. 对决 → 11. 道德自我揭露 → 12. 道德抉择 → 13. **主题的揭露**（打破角色界限，让观众体悟普世道理）

## 判定标准

- 主角是否有**道德层面**的需求（而不只是心理层面）——测试：主角在故事开头至少对一个人造成了伤害
- 每个角色是否都是核心道德难题的**不同变奏**
- 主题是否通过**结构**（对决/自我揭露/抉择的时刻）传达，而非通过对白说教
- 主角的**道德抉择**是否面临真正的两难（两正或两负，而非正vs负）
- 对手是否有**有力的辩白理由**（避免善恶简单对立）
- 道德论点的揭露是否给观众带来"恍然大悟"的冲击

## 常见错误

1. **只通过对白传达主题**：角色变成主题的传声筒，故事变得说教。
2. **主角只有心理层面的需求**：缺乏道德需求会限制角色影响范围——他的行动只影响自己。
3. **让对手成为"稻草人"**：对手的论点软弱无力，导致善恶对立过于简单。
4. **道德抉择不够艰难**：在正vs负之间选择，观众一眼看出答案。
5. **主题的揭露过早**：悬念消失，故事中段疲软。
6. **道德论点与情节失衡**：情节不足时道德论点变成说教；情节过强则道德论点失去力量。

## 交叉引用：McKee 控制意念

| 维度 | Truby 道德论点 | McKee 控制意念 |
|------|---------------|---------------|
| 本质 | 以行动表现的道德序列 | 价值观变化的陈述句 |
| 表达形式 | 结构收敛→主题扩展的过程 | "XX导致XX"的价值判断 |
| 与角色关系 | 每个角色是主题的变奏 | 主角的最终选择揭示主题 |
| 对手角色 | 对手必须有有力的辩白 | 对抗力量总和测试主角价值 |
| 对道德复杂性的处理 | 四角对立vs善恶二元 | 正面vs负面价值的渐进变化 |
| 具体工具 | 道德论点策略序列(14步) | 价值转换图/鸿沟分析 |

**互补使用**：用McKee的控制意念明确故事的价值核心（一句话主题），用Truby的道德论点序列设计出通过行动传达主题的完整过程。
