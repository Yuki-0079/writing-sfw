# Skill: 故事写作大师班（John Truby Knowledge Base）

## 概述

本 skill 基于约翰·特鲁比（John Truby）《故事写作大师班》（*The Anatomy of Story*）的结构化知识库，提供**有机故事创作方法论**的诊断与指导。

### 核心定位

特鲁比的理论体系与麦基互补，侧重以下维度：
- **有机故事观**：故事不是机器而是生命体，各子系统相互依存
- **角色网络**：通过角色之间的对立关系网络来定位每个角色
- **设计原则**：用一句话定义故事的独特形式和深层过程
- **22步结构**：从主角弱点/需求到新平衡点的完整情节地图
- **道德论点**：通过结构而非说教传达主题

## 核心框架坐标

| 维度 | 坐标值 |
|------|--------|
| 媒介偏重 | hybrid（适合所有叙事媒介） |
| 分析粒度 | sequence/act（序列/幕级） |
| 分析目标 | structure/character/craft（结构/角色/工艺） |
| 理论刚性 | prescriptive（有明确的步骤和方法论） |
| 通用性 | broad（覆盖故事全部子系统） |

## 可用知识库

- `references/knowledge-base.json` — 75条结构化知识条目（11章全覆盖）
- `references/deep/*.md` — 核心概念的深度分析（含与麦基的交叉引用）

## 核心概念体系

### 七大关键步骤
故事骨架的七个发展阶段：弱点/需求→欲望→对手→计划→对决→真实自我揭露→新的平衡点。详见 `references/deep/七步结构与22步结构.md`

### 22步结构
在七大步骤基础上扩展的完整情节地图，特别为"故事中段"提供15个详细步骤。步骤数量可伸缩。

### 设计原则
故事的种子和内在逻辑——「设计原则 = 故事发展过程 + 独创性的手法」。区分于故事前提（具体情节）和类型（通用公式）。

### 角色网络
通过角色之间的相互比较和定位来塑造角色。「角色往往由他不是什么样的人来界定」。

### 道德论点
通过结构（而非说教）传达的、以行动表现的主题序列。每个角色是核心道德难题的变奏。

### 戏剧符码
深植于人心的成长符码——「我欲，故我在」。欲望驱动的变化是所有故事的底层逻辑。

## 使用指南

### 分析作品时的流程

1. **感知层**：识别作品媒介、结构特征（线性/迂回/螺旋/分支/爆发）
2. **框架激活**：判断 Truby 理论的相关度
3. **多视角诊断**：
   - 从七大关键步骤分析结构完整性
   - 从角色网络分析角色定位
   - 从道德论点序列分析主题传达
   - 从22步结构分析情节设计
4. **交叉参考**：与 McKee 框架的差异处进入辨析模式

### 辨析模式

当特鲁比与麦基的理论对同一问题有分歧时：
1. 陈述双方论据（引用KB原文）
2. 说明分歧根源（通常是媒介偏重或理论假设不同）
3. 给出基于作品特征的倾向性建议
4. 邀请创作者讨论

### 三语气规则

- **鼓励模式**：「从七大关键步骤的角度看，你已经在弱点/需求环节打下了很好的基础……」
- **客观模式**：「22步结构中，你目前完成了欲望→对手→计划，但还缺少'看似落败'这一关键节点。」
- **尖刻模式**：「你的对手定义错了——他不是与主角竞逐同一目标的人，所以故事缺乏真正的冲突。」

## 边缘情况处理

- **用户指定只用特鲁比**：禁用麦基参考，纯用22步/角色网络/道德论点体系
- **作品与特鲁比理论不符**（如高度实验性叙事）：坦诚说明，转通用有机故事原则
- **22步框架冲突**：提示22步是可伸缩的，根据篇幅调整步骤数量
