# 价值转变图谱 (Value Shift Catalog)

## 麦基核心定义

价值（Value）在麦基的叙事理论中，指的是故事人物的核心处境——从一种状态到另一种状态的"二元对立"的价值转变。

**麦基原话：** "一个故事事件在人物的人生处境中创造了一个有意义的改变，这种改变是用价值来表现和体验的，并通过冲突来达成。"

## 价值转变的核心原理

1. **每一个场景都必须有价值转变** —— 如果一个场景从开始到结束，人物的核心处境没有发生变化（正↔负），这个场景就不应该存在
2. **价值在序列中递增转变** —— 从场景→序列→幕→故事，价值的转变幅度越来越大
3. **高潮中的价值转变必须是不可逆的** —— 故事结尾，人物不能回到起点的价值状态
4. **正面价值与负面价值之间有"否定之否定"** —— 在正负两极之间，存在一个比单纯的"负"更糟的"否定之否定"状态

## 价值转变在不同叙事层级上的表现

| 层级 | 转变幅度 | 示例 |
|------|---------|------|
| **节拍** | 微小的行动/反应转向 | A 挑衅 → B 反击 |
| **场景** | 一个清晰的正↔负转向 | 希望 → 绝望 |
| **序列** | 2-5 个场景合并后的更大转向 | 计划执行 → 计划崩溃 |
| **幕** | 一个主要的叙事转向 | 第一幕：设定 → 第二幕：对抗 |
| **故事** | 不可逆的终极价值转变 | 自由 ↔ 囚禁（不可逆） |

## 价值转变检测方法

### 场景级检测
```
场景开始：人物的价值状态是什么？（正/负，具体是什么价值？）
     ↓
场景结束：人物的价值状态是什么？
     ↓
两个状态相同？
  ↙ Yes        ↘ No
场景需要重写    价值转变发生了
               ↓
          转变有说服力吗？
         ↙ Yes    ↘ No
      场景成立   需要强化中间节拍
```

### 故事级检测
1. 在故事开头标记主人公的核心价值状态（比如：被囚禁在自我欺骗中）
2. 在高潮时标记核心价值状态（比如：从自我欺骗中解放，但付出了代价）
3. 检查这两个状态是否为**同一价值的正负两端**且**不可逆**
4. 检查故事中的每个场景是否都在这场"从A到B的价值旅程"中提供了关键的转变步骤

## 常见价值转变错误

### 1. 零转变场景 (Zero-Change Scenes)
**症状：** 场景开始和结束时人物的价值状态完全相同
**麦基："一个场景如果没有价值转变，这个场景就不应该存在"**
**修正：** 要么重写场景加入价值转变，要么删除场景

### 2. 反向转变 (Reversal Without Grounding)
**症状：** 价值从一个方向突然跳转到相反方向，没有充分铺垫
**修正：** 确保每一个价值转变都有因果链支撑

### 3. 可逆高潮 (Reversible Climax)
**症状：** 故事高潮中的价值转变，理论上可以"撤销"
**麦基："高潮必须是不可逆的——一旦发生，就永远改变了故事的宇宙"**

### 4. 价值转变过于表面
**症状：** 转变仅仅发生在情节层面（赢了/输了），没有触及人物内在
**修正：** 确认外在的情节转变是否同时引发了人物内在的价值再评估
