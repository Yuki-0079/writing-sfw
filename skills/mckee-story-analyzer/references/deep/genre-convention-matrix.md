# 类型约定矩阵 (Genre Convention Matrix)

## 麦基核心论断

**"类型是观众和作者之间的契约。"** 每一个类型都设定了一套特定的期待，作者必须满足这些期待，同时在此基础上创新。

**麦基：** "如果完全绕过这些期待，观众不会感到惊喜——他们会感到被欺骗。创新不在于推翻类型，而在于在类型内找到尚未被发现的视角。"

## 15 种类型 × 5 维度的约定矩阵

### 矩阵说明
- **情感体验：** 观众/读者应该获得的核心情感
- **必须场景：** 这类故事必然包含的典型场景
- **典型人物：** 常见的角色原型
- **冲突类型：** 典型的冲突形式
- **世界规则：** 世界的运作方式

### 1. 爱情故事 (Love Story)

| 维度 | 约定 |
|------|------|
| 情感体验 | 心动的甜蜜 → 分离的痛苦 → 团圆的喜悦 / 永别的悲伤 |
| 必须场景 | 初次相遇、第一次约会/互动、分离/阻碍、重逢/高潮 |
| 典型人物 | 恋人双方、阻碍力量（家庭/社会/第三方）、支持者（闺蜜/哥们/导师） |
| 冲突类型 | 人与社会（阶级/家庭反对）、人与人（错误认知）、人与自我（恐惧承诺） |
| 世界规则 | 现实世界规则，但爱情的"命运感"可以超越日常逻辑 |

### 2. 恐怖 (Horror)

| 维度 | 约定 |
|------|------|
| 情感体验 | 逐渐积累的不安 → 惊吓 → 最终是释然/绝望 |
| 必须场景 | 黑暗环境、独处时刻、怪物/杀手的追踪/发现、致命对抗 |
| 典型人物 | 受害者、怪物/杀手、幸存者（通常是"最终女孩"） |
| 冲突类型 | 人与超自然/疯狂/死亡 |
| 世界规则 | 与现实相似的底层逻辑，但被恐怖力量颠覆。恐怖力量的规则必须一致 |

### 3. 成长故事 (Coming-of-Age / Maturation Plot)

| 维度 | 约定 |
|------|------|
| 情感体验 | 从天真到觉醒的"酸涩的甜蜜" |
| 必须场景 | 天真的展示、失去天真的事件、尝试回归（失败）、接受成长 |
| 典型人物 | 成长中的主角、父母/导师（被推翻/死亡的权威）、朋友（同行者或分道扬镳者） |
| 冲突类型 | 人与自我（内在认知的转变）为主，人与社会为辅 |
| 世界规则 | 现实世界。核心规则：一旦"看见"，就无法"不看见" |

### 4. 悬疑/侦探 (Mystery / Crime)

| 维度 | 约定 |
|------|------|
| 情感体验 | 好奇 → 线索的拼凑 → 揭示的震惊和满足 |
| 必须场景 | 发现真相的开端、调查/追踪、误导/假线索、真相揭示 |
| 典型人物 | 侦探/调查者、受害者、嫌疑人、真凶 |
| 冲突类型 | 人与人的智力对弈，真相与谎言的对抗 |
| 世界规则 | 现实世界。核心规则：所有线索必须在揭示前已经存在（公平游戏） |

### 5. 动作/冒险 (Action/Adventure)

| 维度 | 约定 |
|------|------|
| 情感体验 | 兴奋、紧张、释放的循环 |
| 必须场景 | 动作场景序列、对抗力展现、决战 |
| 典型人物 | 英雄、反派、导师/伙伴、待拯救的对象（人或物） |
| 冲突类型 | 人与外部力量的身体冲突 |
| 世界规则 | 允许高于现实的物理/武艺能力，但必须在内部保持一致性 |

### 6. 喜剧 (Comedy)

| 维度 | 约定 |
|------|------|
| 情感体验 | 笑的释放 → 最终是人性的温暖/秩序回归 |
| 必须场景 | 常态被打破、混乱升级、误解/错位、回归秩序 |
| 典型人物 | 喜剧主角（通常有执念或缺陷）、直人（正常人对比）、障碍角色 |
| 冲突类型 | 人与社会/体制的荒诞冲突，人与自我（执念）的内向冲突 |
| 世界规则 | 现实被扭曲但保持内在逻辑。喜剧的规则是：错误的事情在错误的时间发生在错误的地方 |

### 7. 悲剧 (Tragedy)

| 维度 | 约定 |
|------|------|
| 情感体验 | 恐惧与怜悯 → 情感净化（Catharsis） |
| 必须场景 | 主人公的优势展示、性格缺陷的暴露、不幸的连环、最后的毁灭/堕落 |
| 典型人物 | 悲剧英雄（有致命缺陷的伟人）、对手（触发者）、见证者 |
| 冲突类型 | 人与自我的内在冲突（核心），外在冲突是内在的投影 |
| 世界规则 | 因果不饶人——每一个选择都会产生后果，无法挽回 |

### 8. 战争/史诗 (War/Epic)

| 维度 | 约定 |
|------|------|
| 情感体验 | 宏大感 → 人性在极限中的暴露 |
| 必须场景 | 战争前奏、首次交战、关键的逆转战役、战后反思 |
| 典型人物 | 士兵/战士、指挥官、平民（被战争影响的人）、敌人 |
| 冲突类型 | 人对抗人与体制、人对抗命运 |
| 世界规则 | 历史或类历史世界。战争的逻辑：代价永远超出预期 |

### 9. 科幻 (Science Fiction)

| 维度 | 约定 |
|------|------|
| 情感体验 | 惊叹感（Sense of Wonder）→ 对人性/社会的反思 |
| 必须场景 | 世界/科技展示、"如果...会怎样"问题的具体展现、科技/体制的后果 |
| 典型人物 | 探索者/科学家、被技术异化的人、体制代表人、普通人视角 |
| 冲突类型 | 人对抗技术/体制、人对抗被技术改变的人性 |
| 世界规则 | 可以有超越现实的规则，但必须建立完整的规则体系并前后一致 |

### 10. 奇幻 (Fantasy)

| 维度 | 约定 |
|------|------|
| 情感体验 | 惊奇与魔法感 → 善恶对抗中的道德澄明 |
| 必须场景 | 魔法/奇异世界的入门、旅途/任务、力量的获得与代价、终极对抗 |
| 典型人物 | 英雄/被选中者、导师、伙伴、黑暗君主/反派 |
| 冲突类型 | 人与超自然力量、善与恶、内心的光明与黑暗 |
| 世界规则 | 魔法的规则必须建立并遵守。力量的获得总是伴随着代价 |

### 11. 教育情节 (Education Plot)

| 维度 | 约定 |
|------|------|
| 情感体验 | 对主人公价值观转变的同情与共鸣 |
| 必须场景 | 旧价值观的展现、深刻的冲击性事件、挣扎、新价值观的接受 |
| 典型人物 | 被教育的主角、催化剂人物（激发转变的人）、障碍人物 |
| 冲突类型 | 人与自我（价值观的内在冲突） |
| 世界规则 | 核心就是："世界观可以被一个事件彻底改变" |

### 12. 社会剧 (Social Drama)

| 维度 | 约定 |
|------|------|
| 情感体验 | 愤怒/无力感 → 对社会结构的揭示 → 希望/绝望 |
| 必须场景 | 体制的展示、小人物的抗争、体制的反击、结局（胜利/失败/妥协） |
| 典型人物 | 小人物/局外人、体制代表人、同盟、旁观者 |
| 冲突类型 | 人对抗体制/社会——这是唯一的核心冲突 |
| 世界规则 | 现实世界。体制的核心：它是无形的、无处不在的、不以个人意志为转移的 |

### 13. 惊悚 (Thriller)

| 维度 | 约定 |
|------|------|
| 情感体验 | 持续的紧张和悬念，直到"真相大白的释放" |
| 必须场景 | 危险的逼近、猫鼠游戏、揭露真相的危险、终极对决 |
| 典型人物 | 主角（普通人被卷入/执着的追寻者）、反派、线人/告密者 |
| 冲突类型 | 人与隐藏的力量，信任与背叛 |
| 世界规则 | 现实世界的底层逻辑被阴谋/秘密织成一张无形的网 |

### 14. 伪纪录片/纪实风格 (Mockumentary / Docufiction)

| 维度 | 约定 |
|------|------|
| 情感体验 | "真实感"带来更亲近的共鸣 |
| 必须场景 | 建立纪录风格的形式、打破第四面墙、现实比虚构更荒诞的时刻 |
| 典型人物 | 纪录片镜头中的"真实"人物、镜头后的人 |
| 冲突类型 | — 从属于其所嵌套的类型 |
| 世界规则 | 假装是"真实世界"。核心规则："我们正在被拍摄"必须在场景层面保持一致 |

### 15. 先锋/实验叙事 (Experimental / Avant-Garde)

| 维度 | 约定 |
|------|------|
| 情感体验 | 以智识上的刺激/困惑替代传统情感体验 |
| 必须场景 | — 无固定场景类型（这正是其"实验"本质） |
| 典型人物 | — 人物可能被抽象化为符号/概念 |
| 冲突类型 | 形式与内容的冲突、读者与世界观的冲突 |
| 世界规则 | 可以故意颠覆因果律、时空间、身份等传统规则。核心约定：**从第一个镜头/句子就要告诉观众/读者"这不是一个常规故事"** |

## 类型混搭的判断原则

| 混搭方式 | 是否允许 | 条件 |
|---------|---------|------|
| 从开头明确宣布混搭 | 允许 | 如："浪漫喜剧"、"科幻惊悚"——从一开始就设定两种体验的混合预期 |
| 类型先后出现 | 谨慎 | 如：前半是侦探故事，后半突然变成超自然恐怖 → 违约（除非在类型选择时已埋下伏笔） |
| 次类型嵌套 | 允许 | 主类型 + 次类型的特定元素（如爱情线嵌套在任何类型中） |
| 反类型 | 允许 | 故意颠覆类型约定，但必须是刻意为之的效果，而非偶然的"失败" |

## 类型创新 vs 违约的判定

| 行为 | 是创新 | 是违约 |
|------|--------|--------|
| 不出现经典场景 | 提供了更有效的替代场景 | 什么也没提供，留下了空白 |
| 改变情感体验 | 升华了类型的情感体验 | 完全没有交付承诺的情感 |
| 不满足常规结局 | 在类型框架内找到了更高明的结局 | 随便收尾，没有解决类型的核心价值问题 |

**麦基最终标准：** 读者/观众看完后的第一反应是"我得到了我想要的，但以我想不到的方式" → 创新。如果是"这不是我买票来看的东西" → 违约。
