# Skill: 资本论 第一卷 — 政治经济学知识库

## 概述

本 skill 基于卡尔·马克思《资本论》第一卷（人民出版社1972年版，中共中央马恩列斯编译局译）的结构化知识库，提供 **马克思主义政治经济学** 的概念解析与 **虚构世界构建应用** 指导。

### 核心定位

本知识库的独特价值在于：

- **经济基础理论**：系统性地阐述商品、货币、资本、剩余价值等核心经济范畴的本质和内在关系
- **辩证法方法论**：从抽象到具体、从简单到复杂的分析路径，可作为虚构世界构建的分析工具
- **批判性世界构建**：为赛博朋克、反乌托邦、后稀缺社会等科幻/奇幻设定提供经济基础设计的理论基础
- **文学分析交叉**：与创意写作理论结合，分析作品中经济关系对叙事结构、人物动机、社会冲突的塑造作用

## 核心概念体系

### 商品与价值理论（第一篇）

| 概念 | 难度 | 关键要点 |
|:-----|:----:|:---------|
| **商品** | ★☆☆ | 用于交换的劳动产品，使用价值与价值的统一体 |
| **使用价值** | ★☆☆ | 物的有用性，财富的物质内容 |
| **交换价值/价值** | ★★☆ | 交换价值是形式，价值是内容——抽象人类劳动的凝结 |
| **社会必要劳动时间** | ★★☆ | 价值量的决定尺度 |
| **劳动二重性** | ★★★ | 具体劳动创造使用价值，抽象劳动形成价值——理解政治经济学的枢纽 |
| **价值形式四阶段** | ★★★ | 简单→扩大→一般→货币：揭示货币的起源和本质 |
| **等价形式三特点** | ★★★ | 使用价值成为价值形式、具体劳动成为抽象劳动、私人劳动成为直接社会形式 |
| **商品拜物教** | ★★★ | 商品形式将社会关系反映为物与物的关系——马克思批判理论的核心 |

### 货币理论（第一篇）

| 概念 | 难度 | 关键要点 |
|:-----|:----:|:---------|
| **价值尺度/价格标准** | ★★☆ | 货币的两种不同职能，马克思对货币理论的贡献 |
| **商品形态变化（W-G-W）** | ★★☆ | 为买而卖——"商品的惊险的跳跃" |
| **买与卖的分离** | ★★☆ | 危机可能性的抽象形式 |
| **货币流通规律** | ★★☆ | 流通手段量 = 商品价格总额 / 货币流通速度 |
| **铸币与纸币** | ★★☆ | 纸币是金的符号，发行限于金流通量 |
| **货币贮藏** | ★☆☆ | 质的无限性与量的有限性的矛盾 |
| **支付手段** | ★★★ | 信用货币的根源，从计算货币到坚硬的货币的突变 |
| **世界货币** | ★★☆ | 金银恢复贵金属形态，国际支付手段 |

### 资本与剩余价值理论（第二~五篇）

| 概念 | 难度 | 关键要点 |
|:-----|:----:|:---------|
| **资本总公式（G-W-G'）** | ★★☆ | 为卖而买，价值增殖是目的 |
| **劳动力商品** | ★★★ | 货币转化为资本的关键——自由的工人 + 一无所有的工人 |
| **劳动过程/价值增殖过程** | ★★★ | 劳动力使用价值创造的价值 > 劳动力本身的价值 |
| **不变资本/可变资本（c/v）** | ★★★ | 马克思的独创区分，揭示剩余价值的真正来源 |
| **剩余价值率（m/v）** | ★★★ | 劳动力受资本剥削的准确程度 |
| **绝对剩余价值** | ★★☆ | 延长工作日 |
| **相对剩余价值** | ★★☆ | 缩短必要劳动时间 |
| **工作日斗争** | ★★☆ | 资本与劳动权力的对抗 |
| **协作→分工→机器大工业** | ★★★ | 资本主义生产方式发展的三个历史阶段 |

### 工资理论（第六篇）

| 概念 | 难度 | 关键要点 |
|:-----|:----:|:---------|
| **工资的本质** | ★★☆ | 劳动力价值或价格的转化形式，掩盖了必要劳动与剩余劳动的区分 |
| **计时工资** | ★☆☆ | 按劳动时间付酬，资本家可灵活调整劳动强度和长度 |
| **计件工资** | ★★☆ | 计时工资的转化形式，使工人自己负责质量和强度 |
| **工资的国民差异** | ★★☆ | 劳动力价值包含历史和道德因素 |

### 积累理论（第七篇）

| 概念 | 难度 | 关键要点 |
|:-----|:----:|:---------|
| **简单再生产/扩大再生产** | ★★☆ | 从再生产视角揭示资本主义生产关系再生产 |
| **所有权规律转化** | ★★★ | 等价交换 → 无偿占有——资本积累的秘密 |
| **资本有机构成** | ★★★ | c:v 比例不断提高——资本主义发展的根本趋势 |
| **相对过剩人口/产业后备军** | ★★★ | 不是人口绝对过剩，而是资本不需要这么多工人 |
| **积累的一般规律** | ★★★ | 一极财富积累，另一极贫困积累 |
| **原始积累** | ★★★ | 不是勤俭节约，而是暴力剥夺——用血与火的文字写入编年史 |
| **剥夺剥夺者** | ★★★ | 资本主义积累的历史趋势——生产社会化与私有制的矛盾 |

## 使用指南

### 适用场景

| 场景 | 说明 |
|:-----|:------|
| **虚构世界经济体系设计** | 使用马克思的分析框架设计虚构社会的货币、贸易、阶级结构 |
| **赛博朋克/反乌托邦批判** | 运用拜物教、积累规律、相对过剩人口等概念分析虚构社会 |
| **文学作品中经济关系的分析** | 从经济基础角度解读人物动机、社会冲突、叙事结构 |
| **经济学理论与文学写作的交叉** | 将政治经济学概念作为批判性世界构建的工具 |
| **与其他写作理论的对话** | 与麦基、systemic-worldbuilding 等框架结合分析 |

### 分析流程

```
1. 感知层：识别作品/设定的经济基础特征
   - 生产形式：商品生产/非商品生产/混合
   - 所有制形式：私有/公有/其他
   - 主要剥削形式：绝对/相对/其他

2. 概念匹配：根据需求选择核心概念
   - 经济体系设计 → 商品/货币/资本/积累
   - 社会批判 → 拜物教/积累规律/原始积累
   - 叙事分析 → 经济关系人格化/阶级斗争

3. 应用映射：将经济学概念转化为叙事/设定元素
   - 查看每个 KB 条目中的 fiction_application 字段

4. 交叉参考：与其他框架对话
   - systemic-worldbuilding：级联后果追踪
   - economic-systems：具体的经济制度设计
   - mckee-story-analyzer：人物动机中的经济维度
```

### 深度阅读路径

```
初学者路径（3个核心概念）：
  商品 → 货币 → 资本

进阶路径（深入理论）：
  价值形式 → 剩余价值 → 资本积累 → 原始积累

世界构建应用路径：
  商品拜物教 → 劳动力商品化 → 积累一般规律 → 剥夺剥夺者
```

| 情况 | 处理方式 |
|:-----|:---------|
| 用户对马克思主义有误解/偏见 | 聚焦于理论本身的方法论价值，不进行意识形态辩论 |
| 需要简化解释 | 使用核心概念区（★☆☆/★★☆/★★★）引导难度选择 |
| fiction_application 不足 | 从 scope_note 和原理解析推导应用方式 |
| 跨框架冲突 | 明确标注视角差异（"从马克思的角度…从主流经济学的角度…"）|
