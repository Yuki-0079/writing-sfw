# 价值形式与货币起源 — 深度分析

> 对应条目：dk-007 ~ dk-011, dk-023 ~ dk-031 | 来源：第一章第3节、第三章

## 核心论点

货币不是被'发明'的，而是商品交换关系发展的自然产物。马克思通过四种价值形式的演化揭示了货币的起源，这是超越古典政治经济学的关键贡献。

## 四种价值形式的演化

```
简单价值形式 → 扩大价值形式 → 一般价值形式 → 货币形式
  (偶然交换)    (经常交换)      (共同等价物)    (固定为金银)
```

### 1. 简单价值形式
x量商品A = y量商品B（如20码麻布=1件上衣）
- 包含一切价值形式的秘密
- 两极：相对价值形式（麻布）和等价形式（上衣）
- 等价形式的三个特点：使用价值→价值、具体劳动→抽象劳动、私人劳动→直接社会劳动

### 2. 总和的或扩大的价值形式
z量商品A = u量商品B 或 = v量商品C 或 = w量商品D...
- 价值本身才真正表现为无差别人类劳动的凝结
- 但仍然是未完成的、永无止境的

### 3. 一般价值形式
一切商品 = 20码麻布（统一等价物）
- 价值形式变得简单和统一
- 商品世界共同活动的结果——社会公认的形式

### 4. 货币形式
一切商品 = 2盎斯金
- 与一般价值形式的唯一区别：金代替麻布作为一般等价物
- 金银天然不是货币，但货币天然是金银

## 货币的五种职能

| 职能 | 说明 |
|------|------|
| 价值尺度 | 为商品世界提供表现价值的材料（观念的货币） |
| 流通手段 | 作为商品交换的媒介（铸币、纸币） |
| 贮藏手段 | 退出流通的货币蓄水池 |
| 支付手段 | 清偿债务（信用货币的基础） |
| 世界货币 | 国际支付、购买、财富转移（金块形式） |

## 常见错误

| 错误 | 正确理解 |
|------|---------|
| 认为货币是国家'发明'的 | 货币是交换过程自发产生的 |
| 认为金银天然是货币 | 金银只是在特定社会关系下才成为货币 |
| 混淆价值尺度和价格标准 | 价值尺度计量商品价值，价格标准计量金量本身 |
| 认为纸币的价值来自国家权威 | 纸币是金的符号，其价值最终受商品流通规律制约 |

## 写作应用

价值形式演化本身就可以作为虚构世界中'货币诞生'的故事框架：

- **第一阶段（物物交换）**：两个文明在边境偶然相遇，以物易物
- **第二阶段（扩大的交换）**：贸易扩大，一种特产（香料、能量块）成为经常交换的媒介
- **第三阶段（一般等价物）**：社会公认某种商品为统一价值尺度
- **第四阶段（货币固定）**：国家将这种商品铸造成标准化的货币

每个阶段的转变都是一个潜在的叙事转折点。
