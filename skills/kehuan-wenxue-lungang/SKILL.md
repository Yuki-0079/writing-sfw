# SKILL: 科幻文学论纲 · 知识库

基于吴岩《科幻文学论纲》的结构化知识库。本书从权力分析视角出发，将科幻文学定位为科技变革时代中边缘人（四类"作家簇"）对社会主流进行的权力解构。

## 核心论纲

科幻小说 = 科技变革时代中，受到社会压制的边缘人（女性/大男孩/边缘人/落伍者四类作家簇）通过作品对社会主流思想、主流文化和主流文学所进行的权力解构。解构方式：**欲望对抗化、内容陌生化、形式方法化、人物种族化**。科幻文学是人类表达**另类现代性蓝图**的某种谋划。

## 知识库使用方式

本知识库包含 53 条结构化知识条目，按章节组织：
- 引子（15条）：权力分析框架 + 中国科幻理论史（梁启超-鲁迅两极、苏联影响）
- 第1章（10条）：科幻的边缘地位 + 三大科幻理论派别

## 分析逻辑

1. **适用场景**：分析科幻文学的社会身份和权力位置，理解科幻"为何如此"而非"应该如何"
2. **核心问题**：这部作品/这位作家的社会边缘性体现在哪里？属于哪类作家簇？
3. **与主流理论的关系**：本知识库不提供创作技法指导（与麦基/特鲁比互补），而是提供对科幻文学社会本质的理论理解
