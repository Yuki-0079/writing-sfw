# 定时写作训练法

## 定义

布兰德设计的两阶段写作训练方法，是全书最著名的实践贡献。核心是通过**两个简单而专横的练习**，从根本上解决写作的心理障碍，培养稳定的创作能力。

## 方法详解

### 第一阶段：清晨写作（训练无意识）

- **做法**：每天早上醒来后，立即开始写作，不读任何文字、不说话
- **内容**：记录任何出现在脑海中的内容——昨夜的梦、前天的活动、真实的或虚构的对话、意识流
- **纪律**：不要重读、不要修改、不要评判
- **目的**：让无意识自由流动，抑制批评家过早介入
- **时长**：持续进行直到能够随心所欲写出流畅文字

### 第二阶段：定时写作（训练纪律）

- **做法**：每天在预先约定的时间开始写作
- **内容**：想写什么就写什么
- **纪律**：无论有没有灵感都要写，不能找任何借口
- **目的**：训练无意识按命令工作，而非等待灵感
- **核心警告**：**"如果你总是做不了这个练习，就放弃写作吧。"**

### 第三阶段：二者合一（完整练习）

- 将两个练习合并：清晨写作 + 定时写作同时进行
- 在此基础上加上：前期准备→设定写作时间→暂时遗忘→到时写作→搁置→批评式重读

## 判定标准

| 成功标志 | 失败标志 |
|----------|----------|
| 能够每天定时产出文字 | 总是找借口推迟 |
| 文笔逐渐流畅 | 坐在桌前发呆、咬笔头 |
| 对写作的抗拒逐渐减少 | 对写作的抗拒大于对写作的愿望 |
| 能够在规定时间进入写作状态 | 只能在有灵感和激情时写作 |

## 常见错误

1. **急于求成**：练习几天后就期待写出完美作品 → 这一阶段的唯一目标只是"写"
2. **写作前阅读**：写作前读小说、报纸、邮件 → 会无意识地模仿所读内容
3. **边写边改**：写作时不断重读和修改 → 批评家过早介入，阻断无意识
4. **不坚持**：中断几天后觉得"断了就重新来" → 习惯需要连续性
5. **自我诊断错误**：认为"今天写得不好"就放弃练习 → 写得好坏不重要，重要的是在写
