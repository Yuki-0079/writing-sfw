> 可扩展框架 · 当前版本：1.0.0（含《资本论》第一卷）

## 概述

本 skill 是 **马克思主义政治经济学** 的系统化知识库，以马克思《资本论》为核心，覆盖商品、货币、资本、剩余价值、工资、积累等核心范畴，并提供虚构世界构建的批判性应用指导。

### 核心定位

| 维度 | 说明 |
|:-----|:------|
| **理论体系** | 辩证唯物主义与历史唯物主义的政治经济学分析 |
| **方法论** | 从抽象到具体、从简单到复杂的辩证法 |
| **应用场景** | 科幻/奇幻世界构建、赛博朋克/反乌托邦批判、文学分析的经济维度 |
| **扩展性** | 多卷本架构，逐步覆盖《资本论》第二、三卷及其他马克思主义经典著作 |

## 扩展指引

当后续卷完成拆解后，按以下流程添加：

```
1. 在 references/ 下新建 vol{N}/
   ├── knowledge-base.json  （本卷KB）
   ├── overview.json         （本卷框架坐标）
   ├── chapters/             （章节原文）
   └── deep/                 （深度分析）
   
2. 更新 references/index.md  → 添加新卷概念索引
3. 更新 scripts/query_all.ps1 → 更新卷检测逻辑
4. 更新 SKILL.md 的概念体系 → 添加新概念
5. 更新 _meta.json           → 更新版本号与条目统计
```
