# 可争议话题清单

## 1. 三幕结构是普适法则还是文化产物？

**麦基立场**：三幕是故事设计的普遍原理，是观众心理的自然需求。
**特鲁比立场**：七步结构（而非三幕）才是故事的根本形式，幕是人为划分。
**盖利肖立场**：四转折点+三分区，不区分幕。短篇无需幕结构。
**辨析指引**：关键区别在于篇幅——长篇（小说/电影）需要幕结构来管理节奏，短篇（<1万字）用四转折点就够了。当创作者困惑于"我的故事该分几幕"时，应优先回答"我的篇幅需要怎样的节奏管理"而非"什么才是正确的结构"。

## 2. 人物应该先于情节还是情节先于人物？

**麦基**：两者一体两面，人物即情节。人物的选择决定故事的走向。
**特鲁比**：人物是第一位的。从角色弱点出发即可推导出后续行动。
**伯罗薇**：可以从人物出发，也可以从情节出发，两者没有固定的先后顺序。
**盖利肖**：任务是设计"刺激→反应"模式，人物和情节同时存在。
**辨析指引**：关键不是谁先谁后，而是你卡在哪里。如果卡在起点不知道怎么开始→从人物出发（特鲁比法）。如果有了人物但不知道怎么写→从情节结构入手（盖利肖法）。

## 3. 科幻和奇幻的边界在哪里？

**吴岩**：科幻必须处理科学霸权的关系，边缘人的权力反抗是其本质。
**谢开来**：奇幻的核心是超自然和架空世界。科幻更接近认知性陌生化（苏恩文）。
**综合**：两者共享"陌生化"手法，但科幻受科学边界的约束（吴岩称"不自由的想象力"），奇幻的超自然则不受此限。交叉地带（科学奇幻）可借鉴两套框架。

## 4. 写作可以教吗？

**布兰德**：可以。写作技巧可通过训练获得，但前提是解决了心理障碍（意识与无意识的协同）。
**麦基/特鲁比**：可以。故事设计有明确的原理和技法。
**吴岩/谢开来**：可以分析。学术研究虽不教创作，但揭示创作的社会机制。
**辨析指引**：技法可以教，天赋不能教。但大多数问题不是天赋问题，而是"技法未掌握+心理未准备好"的问题。

## 5. 设定文本是否不可替代？

**谢开来**：不可替代。设定为叙事提供框架和一致性，是社会化生产的基础。
**直觉反驳**：很多成功作品没有公开的设定集。
**辨析**：公开的设定集≠设定文本。即使没有公开，作者内部也需要设定文档。失败的"以文代设"案例（九州）证明，纯粹用小说承载设定会导致系统崩溃。
